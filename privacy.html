B站视频笔记 — 隐私政策

最后更新：2026-07-01

1. 我们收集什么

「B站视频笔记」扩展仅在您主动触发视频总结功能时，向服务器发送以下信息：

  • B站视频 URL / BV号 — 用于定位视频和获取字幕
  • B站 Cookie（SESSDATA 等会话标识） — 用于通过 B站 API 验证身份并获取字幕数据
  • 身份码（BN-XXXXXXXX） — 用于识别您的付费账户和信用点余额
  • 笔记样式偏好 — 经典亮点 / 思维体系 / 导航洞察 / 引导式

2. 数据用途

  • Cookie 仅在单次请求中通过 HTTPS 发送至服务端，用于调用 B站官方接口获取视频元数据和字幕文本
  • 服务端 不存储 您的 Cookie，请求完成后即丢弃
  • 身份码用于关联您的信用点账户，完成消费扣费和余额查询

3. 数据共享

  • 字幕文本会发送至 DeepSeek（deepseek.com）大模型 API 进行 AI 总结
  • 无字幕视频的音频转写（ASR）由第三方服务提供
  • 除上述第三方 AI 服务外，您的数据不会出售、分享或用于其他目的

4. 支付信息

本扩展不自建支付系统。付费充值通过第三方支付平台完成，我们不存储您的微信/支付宝账户信息。

5. 数据安全

所有数据传输均通过 HTTPS 加密。服务器数据库仅存储身份码对应的信用点余额和交易记录。

6. 您的权利

您可以随时在扩展设置中查看身份码和余额。如需删除账户或导出数据，请联系我们。
